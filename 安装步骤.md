# ChiBank 安装步骤 - 跟着做就行

## 第1步：检查 Docker

```bash
docker --version
```

**看到版本号？** 继续下一步  
**没有？** 先运行：`curl -fsSL https://get.docker.com | sh`

---

## 第2步：创建文件夹

```bash
mkdir -p ~/chibank && cd ~/chibank
```

---

## 第3步：下载配置文件1

```bash
curl -O https://raw.githubusercontent.com/hhongli1979-coder/chibank999/main/docker-compose.prod.yml
```

---

## 第4步：改名

```bash
mv docker-compose.prod.yml docker-compose.yml
```

---

## 第5步：下载配置文件2

```bash
curl -O https://raw.githubusercontent.com/hhongli1979-coder/chibank999/main/.env.example
```

---

## 第6步：复制配置

```bash
cp .env.example .env
```

---

## 第7步：设置密码

```bash
nano .env
```

找到 `DB_PASSWORD=` 这行，改成你想要的密码，比如：
```
DB_PASSWORD=你的强密码123!@#
```

按 `Ctrl+O` 保存，`Ctrl+X` 退出

---

## 第8步：拉取镜像

```bash
docker-compose pull
```

**要等几分钟，下载镜像中...**

---

## 第9步：启动服务

```bash
docker-compose up -d
```

---

## 第10步：等待20秒

```bash
sleep 20
```

---

## 第11步：初始化数据库

```bash
docker-compose exec app php artisan migrate --force
```

---

## 第12步：完成！

打开浏览器，访问：**http://localhost**

---

## 常用命令

```bash
# 查看日志
cd ~/chibank && docker-compose logs -f

# 停止
cd ~/chibank && docker-compose down

# 重启
cd ~/chibank && docker-compose restart

# 查看运行状态
cd ~/chibank && docker-compose ps
```
